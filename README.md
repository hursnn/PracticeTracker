#Weekly Lesson Note and Practice Tracker
##Overview

This project is a mobile application built using .NET MAUI. The goal of the app is to help music students practice more effectively by keeping track of their daily repetitions and showing their weekly progress.

##Why I Built This

From my experience teaching violin, one of the most important parts of practice is repetition. However, many students either get bored or lose track of how many times they have practiced something. As a result, their practice often becomes inconsistent and less effective.

I wanted to create a simple tool that helps students stay more aware of their practice. By counting repetitions and showing progress, the app encourages more structured and consistent practice habits.

##What the App Does

The app allows users to add and manage students, track daily practice, and view weekly summaries. Students can enter a task, set a target number of repetitions, and tap a button each time they complete one repetition. If they make a mistake, they can undo it.

The app also provides a weekly summary that shows the total number of repetitions, the number of days practiced, and the most practiced task. All data is stored locally on the device.

##Technologies Used

The application is built using .NET MAUI with C#, XAML for the user interface, and SQLite for local data storage.

##How to Run

To run the application, first clone the repository from GitHub. Then open the project in Visual Studio. After restoring the required NuGet packages, build and run the application. It can be run on a Windows machine or an Android emulator.

##Notes

The original idea for this project included a weekly lesson note system where teachers could create and share structured practice plans with students. However, due to time constraints and technical limitations, this version focuses on the core practice tracking functionality.

The current version prioritizes a simple and reliable repetition-based tracking system that helps students build consistent practice habits.

##Future Improvements

Future improvements could include adding a teacher-to-student lesson note system, enabling data sharing between users, implementing import and export features, and adding notifications to remind students to practice. A parent monitoring feature could also be added to help track student progress more closely.
